# Réseaux — Résumé & guide de révision (L3, USTHB)

Pack de révision complet pour le module **Réseaux**, construit à partir des cours,
résumés et **annales corrigées (2017 → 2024)**.

## 👉 Le plus important : le guide de révision

**[`cours-reseaux.html`](cours-reseaux.html)** — un cours complet en un seul fichier,
à ouvrir dans n'importe quel navigateur (rien à installer). Il contient :

- 🧠 une **stratégie d'examen** basée sur 7 années d'annales (ce qui tombe, ce qu'on peut ignorer) ;
- 11 chapitres expliqués simplement, avec **toutes les formules** et des **exemples résolus** tirés des corrigés ;
- 🍳 des **recettes pas-à-pas** pour chaque type d'exercice ;
- 📜 le **détail année par année** de tous les examens ;
- 🃏 des **flashcards** + une **checklist** de dernière minute.

### Les 3 thèmes qui rapportent le plus de points
1. **Adressage IP + sous-réseaux + tables de routage + interconnexion (+ fragmentation)** — ~10/11 examens.
2. **Multiplexage temporel + Modulation** — ~9/11 examens.
3. **Détection d'erreurs** (CRC, VRC/LRC, Hamming) — ~6/11 examens.
